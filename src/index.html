Sitio web de Gradiente y Outline

Proyecto escolar realziado en tiempos universitarios. Es un sitio web a modo tutorial dedicado a las funciones CSS linear-gradient(), radial-gradient(), rgba() y hsla(); asímismo se abarca la propiedad outline. Se pueden crear diferentes marcos exteriores y diferentes gradientes con ambos tipos y N colores haciendo uso de 2 diferentes representaciones de colores.

Diccionario Virtual PHP

Sitio web creado en tiempos universitarios para una asignatura en específico. Este sitio es un diccionario virtual de PHP, donde se pueden guardar palabras reservadas y estructuras propias del lenguaje de programación. Los usuarios que visiten el sitio, pueden agregar ellos mismos las palabras y estructuras, o eliminarlas. Los datos se guardan mediante archivos de texto plano.
